== Description ==

This plugin allows to use real-time shipping rates provided by third-party shipping services.
To create new live shipping service use Shipfare api method [https://ship2api.vndr.com/api/v1/liveShipping/create]).

== Installation ==

1. Upload the `shipfare-live-shipping-for-woocommerce` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Does this plugin require the Woocommerce plugin? =

Yes, the Woocommerce plugin must be installed and active.
